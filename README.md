# 💬 WhatsApp Clone (Full Stack Real-Time Chat Application)

A full-stack real-time messaging application inspired by WhatsApp.  
It supports real-time chat, friend requests, audio/video calling, authentication, and media sharing using modern web technologies.

---

## 🚀 Features

### 💬 Messaging System
- Real-time one-to-one chat
- Instant message delivery
- Message timestamps
- Chat history

### 👥 Social System
- Friend requests (send / accept / reject)
- User search
- Online / Offline status
- User profiles

### 📞 Calling System
- Audio calls 🎧
- Video calls 📹
- Peer-to-peer communication (WebRTC)
- Call signaling via WebSockets

### ⚡ Real-Time Communication
- Socket.IO for real-time messaging
- WebSockets for live events (messages, calls, status updates)
- Live typing indicators (if implemented)
- Instant user status updates

### 📁 Media Sharing
- Image/file sharing
- Cloudinary integration for uploads

### 🔐 Authentication & Security
- JWT authentication
- Password hashing (bcrypt)
- Protected routes (middleware)

### 🧑‍💼 Admin Features
- Admin controls for users
- Seed admin setup
- System management utilities

---

## 🏗️ Tech Stack

### Frontend
- React.js
- Bootstrap / CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Real-Time Layer
- Socket.IO (chat + events)
- WebRTC (audio/video calls)

### Cloud Services
- Cloudinary (media storage)
- Nodemailer (email service)
